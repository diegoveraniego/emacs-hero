⌨ Elige tu tecla Ctrl

En Emacs Hero, una tecla de tu elección actúa como Ctrl. Mantenla presionada y pulsa la segunda tecla para ejecutar comandos C-. Alt sigue siendo Meta (M-).

Esc + f → C-f (forward-char)
Alt + f → M-f (forward-word)
Esc + SPC → C-SPC (set-mark)
